# Special Layers

> 来源: <https://wohlsoft.ru/pgewiki/Special_Layers>

---

In SMBX38A, there are special layer names that have special behaviors. To create a special layer, call the layer by the name. The `*` are wildcards and can have any text inside. use this wildcard to have multiple special layers of the same name. The layer text box will turn red if the layer is named correctly.

[![](/wiki/images/b/b2/Special_Layer_Example.png)](https://wohlsoft.ru/pgewiki/File:Special_Layer_Example.png)

A special layer as shown in the editor. Take note that the layer name is inside the parenthesis and the name is in red.
